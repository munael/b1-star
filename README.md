## Contributing
### Deps:
1. NPM
   ```
   sudo apt install nodejs npm
   ```
2. typescript 3.0+ `npm i -g typescript@latest`
3. types: `npm install` should do it, but if not, do  
`npm install --save @types/<library>`  
Libraris:
   1. `p5`
   2. `w2ui`
   3. `jquery`


### Running locally:
Install `npx`:
```
npm install -g npx
```

Then run:
```
npx serve .
```

Access: `localhost:500/app` (or whatever).
